## Build instructions: 

Dependencies: e.g on Arch Linux: ``sudo pacman -S ncurses go``

1. Clone repository: ``https://github.com/alexanderstephan/gobra.git``
2. Change directory into folder: ``cd gobra``
3. Compile files ``go run *.go``
